# Elliot Pizza

Elliot Pizza - Бот выдуманной пиццерии

## Что добавили

### Основа

- Возможность просмотра меню

- Сделать заказ блюда из меню

- Регистрация пользователя

- Промокоды

- Хранение истории заказов

### Регистрация

- ID Пользователя

- Имя пользователя

- Пароль (Хэширование)

- Email

### Хранение истории заказов

- ID Заказа

- ID Пользователя

- Корзина (Заказы)

- Цена (Без скидки)

- Финальная цена (Со скидкой или без)

- Скидка (%)

- Время

### Примечания

- Данная версия имеет .json Базу Данных - Данная версия была создана тогда, когда автор не знал еще как работать с SQLite3.

- Если нет файлов .json которые отвечают за информации о пользователе, истории заказов и промокодов - то программа сама их создаст 

- Чтобы были скидки надо создать в файле .json о промокодах - сами промокоды
